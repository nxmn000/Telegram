Pertanyaan Umum (FAQ)

Apakah Instant Mail aman digunakan?

Ya, Instant Mail aman digunakan. Ini adalah aplikasi resmi dan kami berkomitmen untuk melindungi privasi dan keamanan data pengguna. Kami menggunakan enkripsi yang kuat untuk melindungi email kamu.

Perangkat seperti apa yang cocok untuk menjalankan Instant Mail?

Instant Mail dirancang untuk bekerja dengan baik di sebagian besar perangkat Android. Persyaratan minimumnya adalah Android 7.0 atau lebih tinggi dan RAM minimal 2GB. Namun, untuk performa optimal, disarankan menggunakan perangkat dengan RAM 3GB atau lebih dan Android 9.0 ke atas.

Cara Install
  1. Unduh file APK dari tombol download di atas.
  2. Buka File Manager Anda dan cari file yang baru saja diunduh.
  3. Klik pada file APK tersebut, lalu pilih Install
  4. Jika muncul peringatan keamanan, pergi ke Pengaturan > Keamanan > Sumber Tidak Dikenal dan izinkan instalasi.
  5. Tunggu hingga proses instalasi selesai dan buka aplikasinya.
